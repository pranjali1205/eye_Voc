Following libraries are Mandatory ; 
Use below insited command line arguments:

In windows:

pyttsx3
$ pip install pyttsx3

pyautogui
$ pip install pyautogui

speech_recognition
$ pip install SpeechRecognition

wmi
$ pip install wmi

AppOpener
$ pip install AppOpener


In Linux/MAC_OS :

pyautogui
$ python3 -m pip install pyautogui

pyttsx3
$ python3 -m pip install pyttsx3

speech_recognition
$ python3 -m pip install SpeechRecognition

wmi
$ python3 -m pip install wmi

AppOpener
$ python3 -m pip install AppOpener


NOTE: If any of the packages aren't insatlling please follow the teminal suggestions and surf for a compatible version of the library for your current python interpeter version
